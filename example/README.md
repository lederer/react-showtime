# Demo site for React Showtime

Created with [Create React App](https://github.com/facebook/create-react-app).

## Development

1.  `yarn link` in `..` to set up a local link for the `react-showtime` package.
2.  `yarn link react-showtime` here to install the locally linked `react-showtime` package.
3.  `yarn start` in `../src/` to watch for local changes to `react-showtime`,
4.  `yarn start` here to watch for changes to the demo site and run a dev server.
